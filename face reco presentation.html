👤 Face Recognition Process

Step-by-Step Workflow:

  • Step 1: Load known face images (Garv, Aashi, Raj, Sanjay, Sudhanshu, Sahil)
  • Step 2: Generate 128-dimensional face encodings for each person
  • Step 3: Capture live video frame from webcam
  • Step 4: Detect face locations in the frame using HOG algorithm
  • Step 5: Generate encoding for detected face
  • Step 6: Compare with known encodings using Euclidean distance
  • Step 7: Match if distance < 0.6 tolerance threshold

😊 Emotion Detection Algorithm

Detection Method

  • Uses DeepFace library with pre-trained models
  • Analyzes facial features and micro-expressions
  • Focuses on 3 primary emotions: Happy, Sad, Neutral
  • Returns confidence scores for each emotion

Key Improvements
